# Credits & lineage

**Terminal Workbench Cursor** is a hard fork of
[cursor-smith](https://github.com/Sadsnake1/cursor-smith) by Sadsnake1 (MIT),
adapted into the Terminal Workbench plugin family.

- Cursor engine, effects, and settings framework originate from cursor-smith.
  Portions © Sadsnake1, MIT — see `LICENSE`.
- The Ghost Trail effect renders the ghost from
  [Terminal Workbench Pet](https://github.com/Real-Fruit-Snacks/terminal-workbench-pet).
- Colors follow the
  [Terminal Workbench design system](https://github.com/Real-Fruit-Snacks/terminal-workbench-design-system)
  (`--twb-*` tokens).
