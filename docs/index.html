Capabilities

Everything the caret can be.

One canvas engine draws every effect, so styles and effects stack without fighting each other — and every bit of motion can be switched off.

Ghost Trail

A miniature Terminal Workbench Pet ghost peels off each character you type — and, if you like, off the caret as it moves. It rises, drifts, and fades.

Box, line & underline

Pick a caret shape, then tune width, opacity, glow, and blink. Optional smooth gliding eases the caret between positions with typing-aware acceleration.

Torch spotlight

Dim the workspace and carry a soft pool of light around the caret or pointer, with adjustable radius, colour, and intensity. Runs alongside any cursor style.

A drawer of effects

Popping letters, a fading pixel trail, motion smear, a cursor trail, and an energy beam — mix and match, each with its own controls.

Terminal Workbench palette

Cursor and ghost colours resolve from the --twb-* accent ramp, so a fresh install looks native under the theme — and falls back to clean defaults without it.

Focused by default

A fresh install shows the ghost and nothing that competes with it. Every effect is individually toggleable, so you decide exactly how much motion the caret has — from dead-still to a full ghost stream.

The Ghost Trail

On by default. Tune it, let it drift.

1

Just type — it's on by default

Ghosts rise from your text out of the box. The controls live under Settings → Terminal Workbench Cursor → Ghost Trail.

2

Choose the triggers

Spawn ghosts as you type, as the caret moves, or both. Density, size, rise speed, drift, lifetime, and opacity are all yours to set.

3

Pick a colour mode

Hold a single accent, cycle the whole palette like a booped pet, or match your cursor colour. Toggle it off any time to go back to a still caret.

Get started

Install in under a minute.

Recommended

Community plugins

  1. Open Settings → Community plugins → Browse.
  2. Search for Terminal Workbench Cursor.
  3. Click Install, then Enable.

Manual

  1. Download main.js, manifest.json, styles.css, and versions.json from the latest release.
  2. Place them in .obsidian/plugins/terminal-workbench-cursor/.
  3. Enable Terminal Workbench Cursor under Settings → Community plugins.

BRAT

  1. Install the BRAT plugin.
  2. Choose Add beta plugin.
  3. Enter Real-Fruit-Snacks/terminal-workbench-cursor.
  4. Enable Terminal Workbench Cursor.
